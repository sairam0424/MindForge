# MindForge v8.0.0 — Celestial Orchestration

**Unified Intelligence, Federated Security, Autonomous Evolution.**

MindForge is an enterprise-grade **Agentic OS** designed to stabilize, secure, and scale autonomous AI coding agents. By transitioning from localized file-based states to a **Federated Intelligence Mesh (FMS)** powered by **Unified Persistence (SQLite)**, MindForge v8.0.0 delivers local reasoning speed at a global organizational scale.

---

## 🚀 Installation & Setup

### 📦 Global Installation (Recommended)
Enable system-wide `/mindforge` commands for daily engineering:
```bash
npm install -g mindforge-cc@latest
```

### 📂 Local Project Seeding
Initialize MindForge in an existing repository with specialized agent identities:
```bash
npx mindforge-cc@latest --local
```

### ⚡ Quick Start (On-the-fly)
Run MindForge immediately for a specific runtime without permanent installation:
```bash
npx mindforge-cc@latest --claude --global
```

### ✅ Verification
After installation, verify your local intelligence health:
```bash
/mindforge:health
```

## 🛠 Core Workflow

MindForge standardizes the agentic lifecycle into four non-bypassable phases:

1.  **`/mindforge:plan-phase`**: Collaborative scope analysis and dependency mapping.
2.  **`/mindforge:execute-phase`**: Parallel wave-based implementation with atomic commits.
3.  **`/mindforge:verify-phase`**: Human-in-the-loop and automated UAT sign-off.
4.  **`/mindforge:ship`**: Changelog generation, quality gate audit, and PR creation.
