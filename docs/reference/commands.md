# MindForge Command Reference

Complete reference for all 36 MindForge slash commands.

## Core Commands

| Command | Description |
|---------|-------------|
| `/mindforge:help` | Display available commands and usage |
| `/mindforge:init-project` | Initialize MindForge in a new project |
| `/mindforge:plan-phase` | Start planning phase for a task |
| `/mindforge:execute-phase` | Execute the current plan |
| `/mindforge:verify-phase` | Verify completed work |
| `/mindforge:ship` | Ship and release workflow |
